Spectrum CSS-Grid Demo

Fixed

Fixed responsive grid.

#element1 {grid-column: 1/4;}
#element2 {grid-column: 4/6;}
#element3 {grid-column: 6/13;}

Fluid

Percent based widths allow fluid resizing of columns and rows.

Layouts in CSS

All you need to remember is grid container, and grid-columns for your layout regions.

<div class="spectrum-grid-row">
  <div class="spectrum-grid-col-xs-12">
    <div class="box">12</div>
  </div>
</div>

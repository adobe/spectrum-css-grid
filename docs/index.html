Nested Grids

Nested grids should be avoided for the most part. Spectrum's grid is intended for layout regions only.

That said, sometimes rules need to be broken. There may be situations where it makes sense to take the grid one level deeper. Here's how you would do it.

First: do not use the spectrum-grid classes to accomplish that. This will create another 12-column grid inside your existing 12-column grid â€“ which will look wierd.

What you really want is a grid that lines up on top of the master grid.

To do that, just apply `spectrum-grid--nested` to any child of the master grid. And then, give it a custom class where you define a `--columns` custom property and a start point on the master grid.

.nested-grid1 {
  --columns: 9; 
  grid-column-start: 1; 
}

.nested-grid2 {
  --columns: 9; 
  grid-column-start: 1; 
}
      
<main class="spectrum-grid">
  <div class="spectrum-grid--nested nested-grid1">
    <div></div>
    <div></div>
    <div></div>
  </div>
  <div class="spectrum-grid--nested nested-grid2">
    <div></div>
    <div></div>
  </div>
</main>

Offsets

Offsetting placement of layout region. Simply define the `grid-column-start` property with the column you want the layout region to begin.

#element1 {grid-column: 10/13;}

Alignment

Align elements to the start or end of a row as well as the top, bottom, or center of a column.

Row alignments are done by specifying `grid-column-start` and `grid-column-end` properties. Column alignment is specified by the `grid-row` property.

Start of row

#element1 {grid-column: 1/7;}

Center of row

#element1 {grid-column: 1/7;}

End of row

#element1 {grid-column: 1/7;}

Top of row

#element1 {grid-row: 1/2;}

Middle of row

#element1 {grid-row: 4/6;}

End of row

#element1 {grid-row: 6/10;}

Distribution

Distributing layout regions is also done by defining `grid-column` start and end properties.

Layouts in CSS

All you need to remember is grid container and layout regions defined in your CSS.

<div class="spectrum-grid spectrum-grid--fluid">
  <div id="element1"> Layout Region </div>
</div>
